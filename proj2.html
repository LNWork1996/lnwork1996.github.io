Python Showcase.

A collection of projects that I created using the Python Programming language.

Brief Description

These are some projects that I've created using Python. Python was the first programming language that I considered myself to be proficient in. I first began using Python before I was a teenager, and after 10 or 12 years, it's still a fundamental language that I often return to.

Job Helper

My most recent project: A way for me to guage the liklihood that I would be a good candidate based off of a job description. This is a fully functioning tool that will read through my resume (once given the file extension) and compare it to a pasted job description that I copied. It will then output a value based on the percent match (see 3.09% in the image). This was designed for personal use and intended to give me an accurate reading on how qualified I am.

Calculator

One of the most basic projects I've created and used as a training exercise. I wanted to begin developing graphical user interfaces using Python, rather than just the simple command prompt that would generate. I figured a calculator was a great step in the right direction because of the buttons, textbox, and how they function together. I created this calculator using PyQt6.

A Personal Wiki Research Tool

This is a tool I created to quickly search Wikipedia. It's proven to be useful when I am researching topics and would like to see multiple entries without the need to open multiple tabs. It's a simple program that could be refined down to only a few lines of code. The best part of this program is that it will display images related to the articles you are researching, as well as the direct URL should you want to look at the article itself.
